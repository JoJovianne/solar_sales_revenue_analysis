# Solar Sales Revenue & ROI Analysis

## PROJECT OVERVIEW
This project simulates a solar financial feasibility analysis for industrial clients in Malaysia.

The objective is to:
-Identify high electricity consumption clients
-Estimate annual solar savings
-Calculate payback period
-Rank clients by financial attractiveness

## BUSINESS CONTEXT
Industrial companies often face high electricity costs and are unsure whether solar investment is financially viable.
This model simulates a financial analysis workflow used by solar consultants to evaluate ROI and prioritize potential clients.

## METHODOLOGY
1. Calculate annual electricity cost
2. Apply different cost savings rates
3. Simulate installation cost
4. Compute payback period
5. Sort clients by ROI

## SAMPLE OUTPUT
The model calculates:
- Annual electricity cost
- Estimated annual savings
- System installation cost
- Payback period (years)

Clients are then ranked by payback period to support sales prioritization.

## SKILLS DEMONSTRATED
- Data cleaning and exploration
- Feature  engineering
- Financial modeling
- Business-driven data prioritization

## KEY BUSINESS INSIGHT
Client with:
- Higher saving rates
- Lower installation cost ratios
Achieve faster payback periods and should be prioritized in sales strategy
